# 事前宣伝型ツイート戦略
作成日: 2026-02-25

## なぜ事前宣伝が現状で機能していないか

現行の「開始前告知」フェーズは **avg 6.0 clicks・CTR 0.34%** と全フェーズ最下位。
原因は構造的なもので、以下の3つが重なっている。

| 問題 | 現状の投稿パターン | 本来必要なもの |
|---|---|---|
| FOMO（今すぐ行動する理由）がない | 「今夜〇〇時から〇〇があります」 | 「なぜ今夜は特別なのか」という物語 |
| URLを踏む動機がない | 時刻と場所を告知するだけ | ブックマークしてでも見に行く理由 |
| 読まれる前にスクロールされる | 1行目が事務連絡 | 1行目が「続きが気になる」状態 |

---

## 事前宣伝が機能する3つの条件

### 条件1: 「ストーリーの続き」として設計する
実況型ツイートは「速報」として機能する。
事前宣伝は「予告編」として機能させる必要がある。

映画の予告編が機能するのは「あらすじ」ではなく「問い」を残すから。
- ✗ `今夜21:00から〇〇銀行強盗があります`
- ✓ `Dear13roが2連敗中の〇〇銀行に、今夜また挑む——今度は成功するのか`

### 条件2: 「今すぐブックマーク」を目的にする
実況ツイートのCTAは「今すぐ見て」。
事前宣伝のCTAは「今すぐブックマーク（後で見るための予約）」に変える。

URLを今すぐ踏まなくていい、でもブックマークしておく理由を作る。
→ マルチビューURLを「当日のリンクをここで確認できる」と位置づける。

### 条件3: 1行目に「誰のどんな状況か」を入れる
時刻や場所より先に、ドラマを置く。

## フォーマット別テンプレート（5型）

### 型1：リベンジ/継続ドラマ型（最推奨）
過去の結果を引用して「今夜の賭け」を演出する。

```
[ギャング名]が前回失敗した[場所]強盗に、今夜また挑む🔥

[前回の結果・状況を1行で]

今夜[時刻]スタート——今度は成功するか、マルチビューで追え
[URL] #[ギャングタグ]
```

例:
```
Dear13roが3週連続で跳ね返された〇〇銀行に、今夜また挑む🔥

前回は逃走中に仲間が…

今夜21:00スタート——今度は成功するか、マルチビューで全員の動きを追え
https://... #Dear13ro
```

---

### 型2：初挑戦/節目回型
「初めて」「最後」「〇回目」などの希少感で興味を作る。

```
[ギャング名]が初めて[場所]に挑む夜🎯

[なぜこの場所/この日が特別なのかを1行で]

[時刻]から——全員の位置がわかるのはマルチビューだけ
[URL] #[ギャングタグ] #NEWTOWN
```

---

### 型3：対立/ライバル演出型
複数ギャングが同時進行する場合の事前煽り。

```
今夜、[ギャング名A]と[ギャング名B]が同時刻に動く⚡

[それぞれの狙いを1行ずつ]

どっちを追う？ マルチビューなら両方同時に見られる
[時刻]スタート → [URL]
```

---

### 型4：予告数字型
「何人」「何分」「何回目」などの具体数字で興味を引く。

```
今夜[時刻]、[人数]人が同時に動く🔥

[ギャング名]の[場所]強盗——マルチビューなら[メリットを数字で]

ブックマーク推奨 → [URL] #[ギャングタグ]
```

---

### 型5：カウントダウン型（当日3時間前に最適）
開始直前の緊張感を事前に作る。

```
あと[X]時間——[ギャング名]の[場所]強盗が始まる

[見どころを1行で・具体的に]

マルチビューリンクはここ → [URL] #[ギャングタグ]
```

---

## 投稿タイミング設計（推奨シーケンス）

実況型（現状最強）と組み合わせる場合の全体シーケンス:

| タイミング | 投稿型 | 目的 | 期待指標 |
|---|---|---|---|
| **前日〜2日前** | リベンジ/節目回型 | 認知・期待醸成 | Imp重視（CTRは低くてよい） |
| **当日3〜4時間前** | カウントダウン型 | ブックマーク誘導 | ブックマーク数 |
| **開始0〜2分** | 開始宣告型（現状最強） | 即時URL流入 | URL Clicks最大化 |
| **開始25〜35分後** | ギャング応援召喚型 | 積み上げ流入 | CTR最大化 |

## 事前宣伝で絶対やってはいけないこと

- ✗ **1行目が時刻・場所だけ**: 事務連絡と区別がつかず、スクロールされる
- ✗ **「よろしくお願いします」系の締め**: 推し活コンテンツに丁寧語は温度感が合わない
- ✗ **前日に4投稿**: 当日の勢いが分散する。事前は1〜2投稿に絞る
- ✗ **URLを文末に埋める**: 事前型はURLの露出を「ブックマーク先」として早めに出す
