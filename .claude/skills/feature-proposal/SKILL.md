# feature-proposal

このスキルは、`/feature-proposal` と同じ目的で使う補助スキルです。  
基本的には `.claude/commands/feature-proposal.md` のルールに従って実行してください。

## 目的
- 新機能案を複数比較できる形にする
- 新規獲得 / 継続利用 / 差別化の観点で優先順位をつける
- 最後に「今着手すべき1つ」を決める

## 出力
- `ideas/[YYYYMMDD]-[topic].md`
- `output/specs/feature-proposals/[YYYY-MM-DD]-[topic].md`
