# 定義

NEWTOWNは、VAULTROOMとCrazy Raccoon系の文脈で進行しているGTA Vの期間限定ストリーマー企画（サーバー企画）です。複数のコミュニティ系まとめ・メディアで、**開催期間は 2026/2/9〜2/28、基本稼働は 18:00〜26:00、月曜休み（定期メンテ）**として整理されています。([v-streamer.playguilds.com][1])

---

# 要点

## 直近の状況まとめ（SNS素材として使える部分）

### 1) いまは「終盤戦」フェーズ

- 期間が **2/28まで**なので、今日（2026/2/24 JST）を含めると残りは **24・25・26・27・28日の5日分**です。終盤の「追い込み」「最終回カウントダウン」文脈が強く使えます。([v-streamer.playguilds.com][1])
- Yahooリアルタイムの `#NEWTOWN` でも、**「最終回のカウントダウンが始まる」** という文言の投稿が上位に出ており、終盤ムードが可視化されています。([Yahoo!][2])

### 2) 公式告知の反応量はかなり大きい（社会的証明として使える）

- 公式告知ポスト（VAULTROOM）の検索スニペット上では、**返信約2.5万件 / 引用約1.9万件 / いいね約17万件 / 表示約2,745万**という規模の反応が出ています（検索時点表示）。([X (formerly Twitter)][3])
- さらに **「2週目スタート」** の公式ポストも、検索スニペット上で **表示約1,244万 / リプ約6,834 / 引用約5,600** と大きく、**中盤以降も熱量が継続**している材料になります。([X (formerly Twitter)][4])

### 3) 直近の話題軸は「ギャング物語」と「切り抜き拡散」

- PANORA（2/17掲載）は、ギャング **Dear13ro.** を深掘りしていて、狂蘭メロコをボスにした“弟子組”の構成（叶、葛葉、イブラヒム、SHAKA系文脈）が整理されています。これは**物語性の強い拡散軸**として使いやすいです。([PANORA][5])
- Yahooリアルタイム（#NEWTOWN）にも、**Dear13ro系の短尺・切り抜き投稿**が直近で流れており、終盤に向けて「関係性」「ギャング進行」を切り抜きで消費する動きが見えます。([Yahoo!][2])
- 別検索（`newtown ぶいすぽ`）でも、**ぶいすぽ×にじさんじの邂逅/会話系切り抜き**が並んでいて、**“初見でも入りやすい会話ネタ”**は依然強いです。([Yahoo!][6])

### 4) 組織・勢力図の変化は、投稿ネタとして強い

- 2/23時点のコミュニティ投稿（Xの検索スニペット）では、**組織一覧更新**として「MondoがSANTA C.L.A.W.S.に加入」といった変化が共有されています。勢力図の変動は、毎日の投稿テーマにしやすいです。※X上のコミュニティ情報なので、投稿時は配信本編/複数視点で再確認推奨。([X (formerly Twitter)][7])

### 5) NEWTOWNの本質は「物語・多視点」

- まとめサイト側でも、NEWTOWNの特徴として **“物語が生まれる街”**、かつ **警察/ギャング/市民など複数視点を比べる楽しさ** が明示されています。StreamerHubのマルチビュー訴求と相性が非常に良いです。([v-streamer.playguilds.com][1])

# 比較

## 情報源の使い分け（マーケ素材用）

### 1. 公式/準公式寄り（信頼度高め）

- **VAULTROOM告知（X）**
  - 取れるもの: 開催告知、節目（2週目スタートなど）、大きな反応量
  - 用途: 投稿の“土台”（開催情報・規模感・権威づけ）
  - 注意: Web上では本文取得が不安定なことがあるので、検索スニペット値は「時点値」扱いが安全。([X (formerly Twitter)][3])

### 2. メディア記事（文脈整理に強い）

- **PANORA**
  - 取れるもの: ストーリー解説、注目チーム/人物の意味づけ、初心者にも伝わる背景
  - 用途: 「なぜ今この話題が盛り上がるのか」の説明文に使える
  - 強み: Dear13ro.のような**具体的な関係性ストーリー**を整理してくれている。([PANORA][5])

### 3. コミュニティ/まとめ（更新速度が速い）

- **atwiki（NEWTOWNまとめwiki）**
  - 取れるもの: 組織ページの詳細、更新時刻、出来事ログ、用語
  - 用途: 今日の投稿ネタの“深掘り材料”
  - 注意: 編集型なので、投稿前に配信/他ソースで裏取りすると安定。
  - 例: `atroce myrtille` ページは **2/24 14:14更新** で、構成/歴史/エピソードが細かく入っています。([アットウィキ][8])

### 4. リアルタイム反応（今なにが刺さっているか）

- **Yahooリアルタイム検索（#NEWTOWN / 関連キーワード）**
  - 取れるもの: 直近の投稿文、切り抜きタイトル、時刻、上位投稿の反応値表示
  - 用途: 今日のポストの言い回し・切り口決め
  - 強み: **「いま何が話されてるか」** の温度感が取りやすい。([Yahoo!][2])

## SNSマーケ用に “今日から毎日” 取るべきデータ項目（ツイート材料生成向け）

> ここは、あなたの「今日の投稿文を作る前処理データ」として使う想定です。

### 1) 話題データ（リアルタイム）

- `#NEWTOWN` の上位投稿文（1〜5件）
- 投稿時刻
- 反応値（Yahooリアルタイムに表示される数値）
- 含まれる固有名詞（配信者名/ギャング名/職業）
- クリップURL有無（YouTube/Shorts）

→ これで「今日の話題語彙」を抽出できます。([Yahoo!][2])

### 2) ストーリー/勢力図データ

- 組織名（ギャング/警察/救急など）
- 新規加入/転職/役割変更
- 主要人物（ボス/中心人物）
- 直近イベント（抗争/逮捕/大型犯罪/練習会）

→ “速報感”のある投稿ネタになります。コミュニティ情報は裏取り前提で。([X (formerly Twitter)][7])

### 3) 公式・規模感データ（社会的証明）

- 公式ポストの表示/反応（時点値）
- 2週目/節目ポストの反応
- 終盤までの日数

→ 投稿の冒頭で「いま熱い理由」を一言で示せます。([X (formerly Twitter)][3])

## StreamerHub向けの実装観点（データ取得の方向性）

NEWTOWN向けの非公式ウォッチャー開発記事では、**YouTube/Twitchの配信データ取得・配信中判定・一覧/検索UI** が要点として挙げられています。これ、StreamerHubのマーケ導線にそのまま使えます。([note（ノート）][10])

### 最低限ほしい取得パイプライン（今日のツイート材料用）

1. **配信中一覧取得**（YouTube/Twitch）
2. **配信タイトル/説明文からキーワード抽出**（例: Dear13ro, 警察, 救急, 練習）
3. **リアルタイム投稿（#NEWTOWN）監視**
4. **1時間ごとに“話題語”ランキング化**
5. **投稿下書き用JSON生成**（あなたのClaude Codeスキルに渡す）
