#### 仮説1: ギャング固有タグ単体の独占的採用による CTR 維持・初期速度最大化

  - `#Dear13ro` 単体: URL Clicks 63.0 / CTR 2.57%（全タグ組み合わせ中1位）
  - `#AtroceMyrtille` 単体: URL Clicks 50.0 / CTR 3.24%（全タグ中最高CTR）
  - `#SANTACLAWS` 単体: URL Clicks 47.0 / CTR 3.91%（全タグ中最高CTR）
  - `#Dear13ro + #NEWTOWN` の2タグ組み合わせ: URL Clicks 37.0（-41%低下）
  - ハッシュタグ研究より、タグ数が増えると「スパム判定リスク」が上昇（Qiita公開コード分析）

- **仮説**: ギャング固有タグを単体で使用することで、複数タグ組み合わせより初期のFor You拡散が強く機能し、ホームミキサーへの入力確率が上昇し、結果としてインプレッション × クリック効率が向上する

- **実験案**:
  - A案（現状パターン）: `#Dear13ro + #NEWTOWN` の2タグ構成で投稿
    - 期待値: CTR 1.63% / URL Clicks 37.0（過去実績）
  - B案（新パターン）: `#Dear13ro` 単体のみで投稿
    - 期待値: CTR 2.57% / URL Clicks 63.0（過去実績）

- **判断指標**: URL Clicks数の絶対値 / CTR / 投稿後30分間のインプレッション伸びスピード（初速）

- **必要サンプル数**: 最低3回。ギャングごとに異なるかを確認するため、Dear13ro・AtroceMyrtille・SANTACLAWS各1回ずつ推奨

#### 仮説3: 開始前告知フェーズの廃止、または「実行中」への統合による投稿数最適化
- **根拠**:
  - 開始前告知フェーズ: URL Clicks 6.0 / CTR 0.34%（全フェーズ最低）
  - その理由: 「時間的猶予がある」→「後で見よう」の選択肢が生まれ、即座のクリック動機が弱い
  - 開始前告知（15分前）から開始直後（0-5分）までに「2つの投稿を打つ」という戦略自体が、アルゴリズムから「連続投稿スパム」と判定されるリスク

- **仮説**: 開始前告知フェーズを廃止し、イベント開始直後1投稿に集約することで、同一イベントあたりの投稿数を削減でき、「連続投稿スパム」判定を回避でき、結果として1投稿あたりのアルゴリズム重み付けが上昇する

- **実験案**:
  - A案（現状）: 開始15分前に告知投稿 → 開始直後に速報投稿（2投稿）
    - 実績: IMP 1,760 + IMP 1,928 = 合計 3,688 / URL Clicks 6 + 34 = 合計 40
  - B案（新パターン）: 開始前告知なし → 開始直後に1投稿のみ（1投稿）
    - 期待値: IMP 1,928以上 / URL Clicks 34以上（同等以上の成果を1投稿で実現）

- **判断指標**: 投稿数あたりの平均URL Clicks / 投稿数あたりの平均CTR

- **必要サンプル数**: 最低2回

### 中期的に検証すべき仮説（2-3回のイベントで検証）

#### 仮説A: Persephone タグの効果反転現象の確認（N=1では判断不可）
- **背景**: 現在Persephoneのデータが極めて限定的（単体1件、2タグ組み合わせ1件）であり、N=1では判断材料として不十分。ただし「#NEWTOWN + #Persephone」は URL Clicks 28 で、単体の14より100%向上しているという唯一の「2タグ>単体」の実績がある

- **仮説**: Persephoneはメジャーギャング（Dear13ro, AtroceMyrtille）と異なり、フォロワー規模が小さいため、単体タグでは到達範囲が狭く、#NEWTOWNタグを組み合わせることで「NEWTOWN大義タグのカテゴリ候補」に昇格し、より広いFor Youリーチが機能する

- **実験設計の要点**:
  - Persephone投稿を最低3回実施。各回で異なるタグ構成（#Persephone単体 / #Persephone + #NEWTOWN / #NEWTOWN + #Persephone順序反転）を試験
  - 各投稿でインプレッション・クリック数を記録し、「タグ配置順序が結果に影響するか」を確認

#### 仮説B: URL Click誘導型 vs Reply誘発型の月間CTR トレードオフ検証
- **背景**: 現在の投稿はほぼ全て「URL Click特化型」（7件中7件が高CTRグループ）で、「返信誘発型」の投稿が0件。Xアルゴリズム観点から「返信（Reply）はURL Click以上に重視される」（推定重み0.4-0.5）との仮説があるが、未検証

- **仮説**: 週1-2本の返信誘発型投稿（問いかけ系：「〇〇のシーン、見た？」など）を混ぜることで、総CTRは若干低下（2.5% → 2.3%）するが、返信→プロフィール経由→URLクリックの「間接的クリック誘導」と For You への推薦拡大により、月間総URL Clicksは1.2-1.5倍に向上する

- **実験設計の要点**:
  - 現在のClick主導型（期待CTR 2.5-3.0%）と、返信誘発型（期待CTR 2.0-2.5%、返信数50+狙い）を同日に混投
  - 返信数・リプライスレッド深掘り・その後のクリック転換を追跡
  - Google Analytics でリファラー「Twitter」の詳細を確認し「直接リンク click」と「スレッド内リプライからの click」を分離できる計測設計が必須

1. **ギャング固有タグ単体戦略** → 最優先。次のイベント（2026-02-26～28推定）から即実施
