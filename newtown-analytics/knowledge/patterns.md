### 解釈（Hook Analyst + X Algorithm観点）

**1位: が始まる/始まった（平均URL Clicks 34.0 / 平均CTR 1.80% / 平均Imp 1,928）**

FOMO（Fear of Missing Out）を最も効率的に刺激するパターン。「始まる」という動詞は視聴者に「今この瞬間を逃すと見られない」という時間的緊迫感を即座に植え付ける。Xアルゴリズムのvelocityシグナル（投稿後数分〜5分前後の初期反応の密度）がFor Youへの候補昇格に直接影響するため、開始0〜2分以内の投稿が必須条件となる。実例：「パレト銀行強盗が始まる」（55 Clicks / CTR 2.22%）、「宝石店強盗が始まる！」（28 Clicks / CTR 2.24%）。同フック内でも明示的CTAがある投稿（55 clicks）とCTAなし投稿（19 clicks）で約3倍の差があり、CTA省略は絶対に避けること。

**2位: サービス紹介（平均URL Clicks 25.6 / 平均CTR 2.34% / 平均Imp 949）**

インプレッション949と低いにもかかわらずCTRが全フック中最高という特異な構造を持つ。ハッシュタグをフックの冒頭に置くパターン（`#Dear13ro マルチビューを使って〜`）がCTR 2.34%を記録。ギャング固有タグが視覚的に「このツイートは誰向けか」を瞬時に伝えるセグメント機能を果たし、ファンが自己同一視しやすい。「応援しよう」という行動提案がクリック行為と応援行為を等価にしている。代表例：`#Dear13ro マルチビューで応援しよう！` (63 Clicks / CTR 2.57%)、`#SANTACLAWS マルチビュー〜` (47 Clicks / CTR 3.91%)。

**3位: その他（平均URL Clicks 20.3 / 平均CTR 1.18% / 平均Imp 1,525）**

投稿数9件と最多カテゴリ。「Dear13roがメイズバンク強盗を始めたみたい」（64 Clicks・全投稿中最高）のように「みたい」という伝聞表現が知的好奇心を刺激するパターンや、「ついに」という副詞が積み重なった文脈を知るコアファンに強く響くパターンが含まれる。内容の質によって個別バラつきが大きいため、平均値より個別事例の型を参考にすること。

**下位フックの判断**

- **修正・報告（廃止推奨）**: CTR 0.04%は全体平均の約30分の1。事務連絡形式は視聴者にとって無関係な情報と判断されてスクロールされる。報告が必要な場合はリプライ限定またはベネフィット訴求に転換する。
- **感情先行（廃止推奨）**: 情報量ゼロの感情語はImp 622（全フック最低）を記録。固有名詞なしの感情語は誰向けの情報かが伝わらない。感情語は「固有名詞 + 感情語」の順序に変える。
- **時刻告知・質問形（改変推奨）**: Impは1,760・1,776と高いがCTRは0.34%に留まる。「見たけど行動しなかった」パターン。時刻告知は1行目に事件名を先行させ時刻を補助情報へ。質問形はURL誘導ではなく返信誘発用途に限定する。

### 転用できるフックテンプレート

1. **即開始型（緊迫感フック）**
   - 型: `[事件名]が始まる👀 URLから[行動訴求]しよう！`
   - 推奨フェーズ: 開始直後（開始0〜2分以内に投稿必須）
   - 期待CTR帯: 高（実績 CTR 1.80〜2.22%・avg URL Clicks 34.0）
   - 根拠: 「が始まる/始まった」フックが平均34.0 Clicks・CTR 1.80%で全フックタイプ中URL Clicks 1位。CTA必須。

2. **ハッシュタグ先行型（ファンダム直撃フック）**
   - 型: `#[ギャング名] マルチビューを使って[ギャング名]を応援しよう！`
   - 推奨フェーズ: イベント紹介・盛り上がり中
   - 期待CTR帯: 高（実績 CTR 2.34〜3.91%・avg URL Clicks 25.6）
   - 根拠: サービス紹介フックが平均CTR 2.34%で全タイプ中最高CTR。`#SANTACLAWS` でCTR 3.91%・`#AtroceMyrtille` で CTR 3.24%と突出。

3. **固有名詞+状況型（実況感フック）**
   - 型: `[ギャング名]が[固有地名/事件名]を[動詞（始めた/動いている/向かった）]みたい👀 URLから[行動訴求]`
   - 推奨フェーズ: 盛り上がり中・結果共有
   - 期待CTR帯: 高（実績最高 64 Clicks・全投稿中最高）
   - 根拠: 「その他」カテゴリのトップ「Dear13roがメイズバンク強盗を始めたみたい」が全投稿中URL Clicks 1位（64 Clicks）。「みたい」という伝聞表現が知的好奇心を刺激。

4. **「ついに/今日だけ」限定感型（特別感フック）**
   - 型: `[ギャング名]がついに[事件名/状況]へ🔥 [特別な理由を1文で] URLから[行動訴求]`
   - 推奨フェーズ: 盛り上がり中・イベント特別回
   - 期待CTR帯: 中（実績 34 Clicks / CTR 1.49%）
   - 根拠: 「Dear13roがついにパシフィック銀行へ。今日は一人のメンバーがお休みに入る特別な一戦」が34 Clicks獲得。「ついに」という副詞が積み重なった文脈を持つコアファンに強く響く。

5. **逃走劇煽り型（実況参加フック）**
   - 型: `[事件名]実行中！！ [状況を1語で煽る（逃走劇/決戦/包囲網など）]を見よう！！👀`
   - 推奨フェーズ: 実行中・進行中（開始後10〜30分）
   - 期待CTR帯: 中（実績 57 Clicks / CTR 1.77%）
   - 根拠: 「飛行場強盗実行中！！ 逃走劇を見よう！！」が57 Clicks。「事件名+実行中」フック平均14.7 Clicksと比較して突出しており、「逃走劇」という臨場感ある1語の追加が差を生んでいる。

### 解釈（Phase Analyst + X Algorithm観点）

**[1位実績値] 結果・余韻（avg URL Clicks 64.0・n=1）**

データ上はURL Clicks最多だがサンプル数が1件のみであり、この投稿（「Dear13roがメイズバンク強盗を始めたみたい」）が実態として「開始直後」フェーズに近い可能性が高い。統計的信頼性の観点では「開始直後」フェーズを最重要フェーズと位置づけるのが妥当。引き続きデータを蓄積して再評価すること。

**[実質1位] 開始直後（avg URL Clicks 34.0 / CTR 1.80% / Imp 1,928）**

「今まさに始まった」という時間的切迫感（FOMO）がユーザーの即時クリック行動を最大化する。Xアルゴリズムのvelocityシグナル（初期反応の密度）がFor Youへの候補昇格に直接影響するため、イベント開始検知から5分以内の投稿が理想。「始まる」→「始まった」への言葉の切り替えタイミングが重要。対照的に「開始前告知」はImp 1,760と高いにもかかわらずURL Clicksが6.0・CTR 0.34%に止まる。アルゴリズム的には「クリックシグナルが弱い投稿」と評価されてFor Youスコアが上がりにくい構造になっている。

**[2位] イベント紹介（avg URL Clicks 25.7 / CTR 1.96%・全フェーズ中最高）**

CTRは全フェーズ中最高（1.96%）であり、少ないインプレッションでも高い確率でクリックに転換できている。ハッシュタグ検索やタイムライン流入で発見したユーザーが「詳しく知るためにリンクを踏む」動線が機能している。インプレッション絶対値は開始直後より低いため、URL Clicks総数の最大化には「開始直後」との組み合わせが必要。

**[最下位] 実行中・進行中（avg URL Clicks 14.7）とサービス告知（avg 1.0）**

実行中は「開始直後」より緊急性が薄れ、「今から入っても盛り上がりが分からない」という心理的障壁が発生する。「まだ間に合う」「今から見ても楽しい」という途中参入ハードルを下げるコピーで設計すること。サービス告知はURL Clicks獲得を目的にすべきフェーズではなく、認知の土台づくりに限定して割り切ること。

**拡散型が0件という課題**

現状の投稿はviral_focused（返信・リポスト誘発型）がゼロサンプル。アルゴリズムのOut-of-Network重み（0.9）を活かし切るには、コミュニティ内での話題生成が必要。月1〜2本の返信誘発型投稿（「誰視点で見る？」「どの枠を入れてほしい？」）をURL誘導の合間に差し込むことでFor Youスコアの底上げが見込める。

### 推奨投稿シーケンス

**2投稿パターン（最小構成）**

1. **開始直後** — 「[事件名]が始まった！URLからマルチビューで見よう」という即時行動喚起コピー — イベント開始から5分以内
2. **実行中・進行中** — 「まだ間に合う。逃走シーンを一緒に見よう」という途中参入ハードルを下げるコピー — 開始から20〜30分後

理由: 最小工数でFOMO（見逃し恐怖）の波を2回作る構成。1本目で即時層を獲得し、2本目でタイムラインを見逃した層を回収する。

**3投稿パターン（標準構成）**

1. **開始前告知** — 「[時刻]から[事件名]が始まる。マルチビューURLはこちら」という事前の動線確保 — イベント開始30〜60分前
2. **開始直後** — 「[事件名]スタート！今すぐURLからマルチビューで見よう」という緊急性・即時性のコピー — イベント開始から5分以内
3. **実行中・進行中** — 「[進捗状況]。まだ間に合う！」という状況報告＋参入誘導コピー — 開始から20〜40分後

理由: 開始前に「認知」を仕込み、開始直後に「行動喚起」し、実行中に「取りこぼしを回収」する3段階。

**4投稿パターン（フル活用）**

1. **イベント紹介** — 「[参加配信者名]が[事件名]に挑む。今日は[特別な要素]がある注目の一戦」という文脈・ストーリーで期待感を醸成 — イベント開始2〜3時間前
2. **開始前告知** — 「[時刻]から開始。URLをブックマークしておこう」というスケジュールと行動の準備を促すコピー — イベント開始30〜60分前
3. **開始直後** — 「[事件名]スタート！URLからマルチビューで一緒に見よう！」という即時行動喚起 — イベント開始から5分以内（最重要投稿）
4. **実行中・進行中** — 「[劇的な展開・経過]。まだ間に合う！逃走劇を見よう」という状況報告＋途中参入誘導 — 開始から30分後または山場のタイミング

各投稿の間隔: 1と2の間は90〜150分（同一アカウントの連続投稿はリーチが落ちる傾向があるため間隔を確保）。2と3の間は30〜60分。3と4の間は20〜40分。最重要ポイントは投稿3（開始直後）のコピー精度と投稿速度。

- [ ] 文字数: フック1行は20〜35文字程度に収め、全体140文字以内。改行を2〜3箇所入れて視認性を確保する
- [ ] URL: 全てのURL投稿にUTMパラメータを付与する（例: `?utm_source=twitter&utm_campaign=newtown_20260226_01`）。直帰率改善のためリンク先を告知・事前型はギャング個別ページ、開始直後・進行中はMultiViewトップページに使い分けることを検討する

1. **クール実況（Cool / Minimalist）**（未検証・最優先）

#### 仮説1: 「が始まる/始まった」フックにギャング固有タグ単体を組み合わせた「速報最強パターン」の再現性検証

### 中期的に検証すべき仮説

#### 仮説A: 「結果・余韻」フェーズ投稿を意図的に設計することで、URL Clicks 40以上を安定して獲得できる

- **背景**: 「結果・余韻」フェーズはn=1（URL Clicks 64 / Imp 3,549）と突出しているが、実態は「開始直後」寄りの投稿内容と推察されており信頼性が低い。意図的に「事件終了後10〜20分以内」に「[事件名]が[成功/失敗]した。マルチビューで全視点のハイライトを見よう」という構成で結果共有型投稿を試す機会がまだない。
- **仮説**: 事件終了直後（結果確定から5分以内）に「結果+余韻+マルチビューでの振り返り誘導」を組み合わせた投稿を出すと、FOMOよりも「感動・共感を追体験したい」という動機でURL Clicksが開始直後と同水準以上になる。
- **実験設計の要点**: 「開始直後」投稿と「結果・余韻」投稿を同一イベントで両方実施し、各URLにUTMパラメータを付与して個別のURL Clicksを比較する。

#### 仮説B: 月2本の「返信誘発型投稿」を継続することでFor Youスコアが向上し、翌週の通常投稿のImpが底上げされる

- **仮説**: 「どのギャングが今回の強盗を成功させると思う？」「次のNEWTOWN、楽しみにしてることは？」のような返信誘発投稿を月2本挿入し続けることで、2〜3週間後にアカウントの通常投稿の平均Impが現在（平均1,325）より10〜20%向上する。
- **実験設計の要点**: 返信誘発型投稿を実施する週と実施しない週で翌週の通常投稿のImpを比較する。期間を最低2週間設けること。返信誘発型投稿自体のURL Clicksは低くなることを前提とし、その後の通常投稿のImp変化を主指標とする。
