### 解釈（Hook Analyst + X Algorithm観点）

**1位: が始まる/始まった（URL Clicks 34.0）**

FOMO（見逃し恐怖）を最も効率的に刺激するパターン。「始まる」という時制の変化が、タイムライン上でニュース速報と同じ心理的重みを発生させる。インプレッション1,928は全フック中最高値であり、リーチと喚起力の両方を兼ね備えた最強パターン。アルゴリズム観点では、投稿後5分前後の反応密度（velocityシグナル）がFor Youへの候補昇格に直接影響するため、開始0〜2分以内の投稿が必須条件。ただし同フック内でもCTAあり投稿（55 clicks・CTR 2.22%）とCTAなし投稿（19 clicks・CTR 0.93%）で約3倍の差があり、CTA省略は避けること。

**2位: サービス紹介（CTR 2.34%で効率首位）**

インプレッション949と低いにもかかわらずCTRが最高という特異な構造を持つ。ギャング固有タグを先頭に置くことで、ファン層への精度の高いリーチが実現している。「マルチビューで応援しよう」という推し活心理と行動喚起が直結しており、クリックまでの心理的障壁がほぼゼロ。ギャング名単体タグとの組み合わせで高CTR（Dear13ro: 2.57%、AtroceMyrtille: 3.24%、SANTACLAWS: 3.91%）を発揮。毎回1投稿は混ぜる戦略が有効。

**3位: 事件名+実行中（URL Clicks 19.0）**

インプレッションは高いがCTRが0.90%と低く、「目に留まるが踏まない」パターンが多い。ただし、CTAを付加した「逃走劇を見よう！！」は57クリック・CTR 1.77%を記録しており、CTA有無で結果が劇的に変わることが判明。このフックを使う場合はCTAを必須条件として設計すること。

**下位フックの判断**

- **修正・報告（廃止推奨）**: CTR 0.04%は突出した最下位。事務連絡は視聴者には関係なくスルーされるのは必然。イベント外の通常時間帯に分離すること。
- **感情先行（廃止推奨）**: 情報量ゼロの感情語はアルゴリズムにも嫌われてインプレッション622（全フック最低）を記録。
- **質問形・時刻告知（改変推奨）**: インプレッションは1,700台あるがCTRが0.34%止まり。具体的な数字・状況・CTA・ギャング固有タグを入れて再設計が必要。

## 4. エンゲージメント構造分類

### クリック特化型パターン（CTR ≥ 2%）

投稿数: 7件 / 平均CTR: 2.94% / 平均URL Clicks: 42.3

**共通要素**:
- ギャング固有タグを先頭または本文前半に配置
- 「マルチビューを使って〇〇を応援しよう！」の定型文またはそれに近いCTA
- URLが文末に1つだけ配置
- 明確なCTA（「URLから同時視聴してみよう！」等）が必ず入っている
- 絵文字は1〜2個で視認性をサポート

**代表例**:
- `#Dear13ro マルチビューを使ってDear13ro.を応援しよう！` → 63 clicks / CTR 2.57%
- `パレト銀行強盗が始まる👀 URLから同時視聴してみよう！` → 55 clicks / CTR 2.22%
- `#AtroceMyrtille マルチビューを使ってatroce myrtilleを応援しよう！` → 50 clicks / CTR 3.24%

### 低パフォーマー傾向

投稿数: 19件 / 平均CTR: 0.77% / 平均URL Clicks: 10.3

**共通する失敗パターン**:
- 行動指示があってもマルチビューの価値が伝わらない曖昧な表現
- 複数ハッシュタグの末尾でCTAとURLが埋もれる構成
- 修正報告・バグ系はURL誘導目的に根本的に不向き
- 時刻告知や感情先行フックで「今すぐクリック」の動機づけがない
- CTAを省いた「が始まる」型（CTAなしでCTRが約3分の1に低下）

#### 仮説1: 「が始まる」フック + 明示的CTA + ギャング固有タグ単体 の組み合わせが URL Clicks 50以上を安定して出す最強パターンになる
