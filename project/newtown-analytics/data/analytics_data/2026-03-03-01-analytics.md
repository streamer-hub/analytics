**「が始まる/始まった」（平均34.0 clicks / CTR 1.80% / Imp 1,928）**

- NEWTOWNとの相性: NEWTOWNはGTA5 RPサーバーであり、強盗ミッションには明確な「開始」「進行」「成否」という物語の流れがある。この文脈では「始まる」が物語の入口として機能し、視聴者が「結末を見届けたい」という欲求でURLをクリックする動機と直結する。配信者にとっても最も盛り上がるフェーズへの誘導であり、マルチビューへの送客効率が高い。

### 下位フックの考察

**「修正・報告」（平均1.0 clicks / CTR 0.04% / Imp 1,298）**

- 不振の理由: 角括弧「[]」で囲まれた修正・バグ報告形式のフックは、視聴者に「自分には関係のない運営側の情報」と即座に判断されてスルーされる。インプレッションは平均1,298と一定数あるにもかかわらず、クリック率がほぼゼロ（0.04%）であることが、コンテンツへの関心のなさを明確に示している。また「[修正報告]」という言葉はSNSのエンタメ文脈において「停止」「問題発生」の印象を与え、配信の盛り上がりとは逆の空気感を作る。
- 改善方針: 捨てるべきパターン。修正・バグ情報はXで発信するならDM・リプライ・問い合わせフォームへ誘導するか、告知が必要な場合でも「マルチビューが今すぐ使えるようになりました」のようなポジティブな言い換えに切り替える。「修正完了」「改善済み」という完了形でサービス価値を前面に出すことで、クリック動機に変換できる。

**「感情先行」（平均2.0 clicks / CTR 0.32% / Imp 622）**

- 不振の理由: 「傑作」のような感情評価語のみを冒頭に置くフックは、何についての「傑作」なのかが瞬時に伝わらない。Xのタイムラインではコンテキストのない一言評価はスルーされやすく、配信を知らないユーザーにとっては完全に内輪の感想に映る。インプレッション自体も622と全フック中最低水準であり、そもそもリーチできていない可能性がある。
- 改善方針: 修正すべきパターン。感情を入口にするなら「何が」「なぜ」傑作なのかを1行目に含める必要がある。例えば「傑作」の一言を捨て、「[シーン名]の[人物]のリアクションが傑作すぎた」のように具体的な状況を添えると、クリック前に価値が伝わる。切り抜き系コンテンツに向いている型であり、NEWTOWNの名場面共有フォーマットとして再設計する余地がある。

**「質問形」「時刻告知」（各平均6.0 clicks / CTR 0.34% / Imp 1,776・1,760）**

- 不振の理由: 両者ともインプレッションは1,700〜1,800台と比較的リーチできているが、クリックに繋がっていない。「質問形」は「ところで初心者女子4人はどう誘われたの？」のような内輪ネタ質問であり、NEWTOWNを知らないユーザーには文脈が届かない。「時刻告知」は「12時40分から！！」という形式で、配信の未来に向けた告知として機能するが、見た時点でその時刻が過ぎていると情報の鮮度が失われ、クリック動機が消える。
- 改善方針: 質問形は「知っている人向けの内輪問いかけ」から「知らない人でも気になる問い」へ転換すること。例えば「[事件名]で何が起きたか知ってる？」のように、答えを見にいきたくなる型に組み替えると効果が変わりうる。時刻告知は単体では弱く、「が始まる/始まった」型と組み合わせて「〇〇時から強盗が始まる」の複合フックに統合するのが現実的。

3. **「キャラクター+状況+緊迫感型」**

5. **「集大成・完結フレーム型」**
   - 型: `[ギャング名]が最後の[イベント名]を始めた👀 URLからマルチビューを見て[最後の/集大成の][シーン]を見守ろう！！`

- **仮説**: 「最後」「集大成」フレームは節目イベントでのみ使うと、CTR 3.5〜4.5%を再現できる。通常投稿に使うと既視感から効果が半減する。
- **実験設計の要点**: 節目イベントか否かを投稿前にメタデータとして記録し、「最後」「集大成」使用の有無と対応するCTR/URL Clicksを蓄積する。10投稿以上のサンプルが集まった段階で比較する。
