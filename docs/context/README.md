# Context

`context/` は、StreamerHub の「土台の認識」を Claude Code に渡すためのディレクトリです。

ここにある内容は、機能案・仕様・マーケティング案のすべての起点になります。

## 含まれる内容

- `service-overview.md`: サービスの目的、対象、主要機能
- `target-personas.md`: 主要ユーザー像と利用動機
- `glossary.md`: サービス内で使う用語の統一定義
- `brand-voice.md`: コピーや説明文で守るトーン
- `value-proposition.md`: ユーザーにとっての価値を整理したもの

## 使い方

- 迷ったらまずここを読む
- 用語の揺れが出そうなら `glossary.md` を基準にする
- 訴求文の方向性は `brand-voice.md` と `value-proposition.md` を優先する
