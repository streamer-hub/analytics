# Value Proposition

## 目的

このドキュメントは、StreamerHub がユーザーに提供する価値を「誰に・何を・なぜ」で整理するためのものです。

## コア価値

StreamerHub のコア価値は、**配信を見つける・比較する・文脈ごと追う**までの距離を短くすることです。

単なる配信一覧ではなく、視聴者の「追い方」を支援することが本質です。

## 提供価値の3本柱

### 1. 追いやすさ

ユーザーは、今どの配信を見ればよいかを判断しやすくなります。

- 関連視点への導線が短い
- イベント単位でまとまっている
- 複数枠の管理負担が低い

### 2. 広がりやすさ

1つの推しや1つの話題から、関連する視点や企画へ広がりやすくなります。

- 推し起点で関連枠に広がれる
- 話題の企画から参加者へ辿れる
- 新しい配信者を見つけやすい

### 3. 理解しやすさ

配信の面白さを「今見ても分かる」方向に補助します。

- イベント構造が見えやすい
- どの視点が重要か判断しやすい
- 断片的な視聴でも流れを掴みやすい

## ペルソナ別価値

### イベント追従コア視聴者への価値

- 同時発生する面白い場面を追いやすい
- 複数視点の比較で、体験密度が上がる
- 企画全体を俯瞰しやすい

### 推し中心ユーザーへの価値

- 推し視点を軸に、周辺情報を取り込みやすい
- コラボの面白さを取りこぼしにくい
- 配信視聴が「1枠だけ」で閉じにくくなる

### ライト層への価値

- 話題の企画を入りやすい形で追える
- サービス価値を短時間で理解しやすい
- 初見でも導線が分かりやすい

### 発見志向ユーザーへの価値

- 今伸びているもの、話題のものに触れやすい
- 関連情報の探索がしやすい
- 視聴範囲を自然に広げやすい

## 競合との差が出やすい価値

### 配信単体ではなく、体験単位で整理できる

一般的な動画検索や配信一覧ではなく、「複数視点」「イベント文脈」「視聴導線」で差が出る。

### 情報整理だけでなく、そのまま視聴行動につながる

まとめサイト的な整理で終わらず、実際に見る行動までつなげやすい。

### 濃い視聴者にもライト層にも意味がある

コア層には比較視聴の深さ、ライト層には導入の分かりやすさがある。

## 絶対に崩したくない価値

- 視聴者の迷いを減らす
- 重要な視点への到達を早くする
- 面白さの文脈を取りこぼしにくくする
- 発見と視聴を分断しない

## 機能提案時のチェック

新しい機能を提案するときは、最低でも次を満たす必要があります。

1. 追いやすさ・広がりやすさ・理解しやすさのどれを強化するかが明確
2. 既存の視聴導線を重くしない
3. コア層だけの内輪機能に閉じない
4. 実際の利用シーンが想像できる

## マーケティングに落とすときの主メッセージ

- 「複数視点で、もっと追いやすく。」
- 「今見るべき配信に、すぐたどり着ける。」
- 「話題の企画を、まとめて見やすく。」
- 「推し視点から、企画全体まで追える。」
