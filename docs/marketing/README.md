# Marketing

`marketing/` は、StreamerHub の集客・訴求・投稿運用の基準をまとめるディレクトリです。

## 含まれる内容

- `channel-strategy.md`: チャネルごとの役割
- `content-pillars.md`: 継続発信するテーマの柱
- `campaign-calendar.md`: 施策を回すサイクルの基本形
- `cta-library.md`: 使い回せる CTA 集
- `hashtag-playbook.md`: ハッシュタグ運用の基本方針

## 使い方

- 投稿案の作成
- イベント時の告知運用
- 紹介動画 / LP の企画
- 施策の優先順位整理
