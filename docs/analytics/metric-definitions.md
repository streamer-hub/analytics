# Metric Definitions

## 目的

このドキュメントは、StreamerHub の主要指標を統一した意味で扱うための定義集です。

## 基本ルール

- 指標名が同じでも、分母分子が曖昧なら使わない
- ダッシュボード・仕様書・振り返りで同じ言葉を同じ意味で使う
- 新しい指標を増やす場合はここに追加する

---

## セッション系

### Session

ユーザーの連続した利用単位。

### Active Session

最低限の意味ある行動（例: 一定滞在、主要ページ閲覧、クリック）が発生したセッション。

### Valuable Session

以下のいずれかを満たしたセッション。

- マルチビュー起動
- 複数視点追加
- イベントページから主要行動に到達

---

## 流入系

### Traffic

全流入数。

### Channel Traffic

特定チャネル（X、検索、直接流入など）からの流入数。

### Event-driven Traffic

イベント関連投稿やイベントページ経由の流入。

---

## 価値体験系

### MultiView Start

マルチビューを起動したイベント。

### Related View Added

既存の視点に、別視点を追加したイベント。

### Core Value Reached

ユーザーが StreamerHub の中核価値に触れたと見なす状態。

推奨条件:
- マルチビュー起動
- または複数視点追加
- またはイベントページ経由の主要行動

### First Value Time

流入からコア価値到達までの時間。

---

## 回遊系

### Page Depth

1セッションで閲覧した主要ページ数。

### Exploration Rate

関連ページ・配信者・イベントへ遷移した割合。

### View Expansion Rate

単一視点から複数視点へ広がった割合。

---

## 継続系

### Revisit Rate

一定期間内に再度訪問した割合。

### Retention

週次 / 月次などの単位で継続利用している割合。

### Returning Valuable Users

過去に価値体験をし、再訪したユーザー数。

---

## マーケティング系

### Post Click Rate

投稿閲覧に対するクリック率。

### Post-to-Core Conversion

投稿経由で流入したユーザーのうち、コア価値到達に至った割合。

### LP Activation Rate

紹介ページやLP閲覧後、コア行動に至った割合。

---

## イベント系

### Event Page Engagement

イベントページ上で主要行動に至った割合。

### Event View Start Rate

イベントページ閲覧から視聴開始またはマルチビュー起動に至った割合。

### Event Return Rate

同一イベント期間中に複数回訪れた割合。

---

## 指標運用の注意

- PV のような表面的な数だけで成功判断しない
- 価値体験系と継続系を必ずセットで見る
- イベント時と平時は分けて比較する
